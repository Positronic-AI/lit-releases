# LIT Platform Releases

Pre-built wheels for the LIT Platform. Install via pip:

```bash
pip install https://github.com/Positronic-AI/lit-releases/releases/download/v0.1.17/positronic_lit-0.1.17-cp313-cp313-win_amd64.whl
```

Replace `cp313` with your Python version (cp310, cp311, cp312, cp313) and platform (win_amd64, manylinux_2_17_x86_64, macosx_11_0_arm64, etc).

## Available Platforms

- **Windows**: win_amd64
- **Linux**: manylinux_2_17_x86_64
- **macOS Intel**: macosx_10_13_x86_64
- **macOS Apple Silicon**: macosx_11_0_arm64
